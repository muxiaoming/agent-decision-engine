# 投资决策接口参数配置说明

## 参数列表

### 必填参数

| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| message | string | 用户投资需求描述 | - |

### 可选参数

| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| modelName | string | AI 模型选择 | deepSeekChatModel |
| threadId | string | 多轮对话线程ID | 自动生成 |
| enableRAG | boolean | 启用知识检索(RAG) | true |
| enableTools | boolean | 启用工具调用 | true |
| enableGraph | boolean | 启用工作流图 | true |

---

## 模型选择 (modelName)

| 模型值 | 说明 |
|--------|------|
| deepSeekChatModel | DeepSeek Chat 模型（推荐） |
| openAiChatModel | MiMo 模型 |
| dashscopeChatModel | 通义千问模型 |

## 功能开关说明

### enableRAG (知识检索)
```java
enableRAG = true   // 从知识库检索财报、市场报告
enableRAG = false  // 跳过 RAG 步骤
```

**使用场景**：
- ✅ 需要基于历史财报分析时开启
- ✅ 需要参考市场研究报告时开启
- ⚠️ 简单查询可以关闭以提高速度

### enableTools (工具调用)
```java
enableTools = true   // 调用股票、指数、收益率等工具
enableTools = false  // 跳过工具调用步骤
```

**可用工具**：
- getStockPrice - 查询股票价格
- getStockHistory - 查询股票历史
- calculateReturn - 计算收益率
- getMarketIndex - 查询大盘指数
- getMarketVolatility - 查询波动率
- getMarketSentiment - 查询市场情绪
- getSectorPerformance - 查询板块表现
- calculatePortfolioReturn - 计算组合收益
- calculateValueAtRisk - 计算 VaR
- calculateSharpeRatio - 计算夏普比率

### enableGraph (工作流图)
```java
enableGraph = true   // 执行完整的工作流程
enableGraph = false  // 跳过工作流步骤
```

**工作流步骤**：
1. 问题感知
2. 知识检索
3. 数据获取
4. 推理分析
5. 决策生成
6. 流程验证

## 请求示例

### 示例 1：基本投资咨询

## 参数优先级

```
1. @RequestParam（查询参数）
2. @RequestBody（请求体）
3. 默认值
```

**示例**：
```bash
# URL 中传递 enableRAG=false
GET /engine/investment/decide/stream?message=xxx&enableRAG=false

# 即使 POST body 中 enableRAG=true，也会使用 false
```

---

## 后端处理逻辑

```java
// 优先使用 @RequestParam
String actualMessage = message;
if (actualMessage == null && body != null) {
    actualMessage = body.message();
}

// 优先使用 @RequestParam
boolean actualEnableRAG = enableRAG;
if (body != null) {
    actualEnableRAG = body.enableRAG();
}
```

## 注意事项

1. **message 是必填参数** - 如果为空，返回错误
2. **其他参数都有默认值** - 可以省略
3. **参数优先级** - @RequestParam 优先于 @RequestBody
4. **多轮对话** - 使用 threadId 保持会话上下文
5. **性能影响** - 功能越多，执行时间越长
