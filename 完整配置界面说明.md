## 配置参数完整列表

### 1. message（必填）

**位置**: 输入框  
**类型**: string  
**说明**: 用户投资需求描述  

**示例**:
```
我想投资科技股，预算10万，风险承受能力中等
```

---

### 2. modelName（可选）

**位置**: 高级选项 → AI 模型下拉框  
**类型**: string  
**默认值**: deepSeekChatModel  

**可选值**:

| 值 | 说明 |
|-----|------|
| deepSeekChatModel | DeepSeek Chat（推荐） |
| openAiChatModel | MiMo 模型 |
| dashscopeChatModel | 通义千问 |

**使用场景**:
- ✅ 需要高质量分析时：`deepSeekChatModel`
- ✅ 需要快速响应时：`openAiChatModel`
- ✅ 需要中文优化时：`dashscopeChatModel`

---

### 3. threadId（可选）

**位置**: 自动生成，无需用户输入  
**类型**: string  
**说明**: 多轮对话的线程ID  

**使用场景**:
- ✅ 需要保持上下文的多轮对话
- ✅ 需要历史记录的场景

**示例**:
```typescript
// 自动生成
const threadId = `investment-${Date.now()}`
```

---

### 4. enableRAG（可选）

**位置**: 高级选项 → 功能开关 → 知识检索 (RAG)  
**类型**: boolean  
**默认值**: true  

**功能说明**:
```java
enableRAG = true   // 从知识库检索财报、市场报告
enableRAG = false  // 跳过 RAG 步骤
```

**使用场景**:

| 场景 | 推荐值 | 说明 |
|------|--------|------|
| 详细投资分析 | true | 需要参考历史财报 |
| 快速数据查询 | false | 跳过知识检索提高速度 |
| 基于文档的问答 | true | 需要检索相关文档 |
| 简单咨询 | false | 不需要知识库 |

**示例**:
```bash
# 开启
GET /engine/investment/decide/stream?message=xxx&enableRAG=true

# 关闭
GET /engine/investment/decide/stream?message=xxx&enableRAG=false
```

---

### 5. enableTools（可选）

**位置**: 高级选项 → 功能开关 → 工具调用  
**类型**: boolean  
**默认值**: true  

**功能说明**:
```java
enableTools = true   // 调用股票、指数、收益率等工具
enableTools = false  // 跳过工具调用步骤
```

**可用工具**:

| 工具 | 功能 | 示例场景 |
|------|------|---------|
| getStockPrice | 查询股票价格 | "查询苹果公司的股价" |
| getStockHistory | 查询股票历史 | "贵州茅台过去一个月走势" |
| calculateReturn | 计算收益率 | "10万投资赚了2万，收益率" |
| getMarketIndex | 查询大盘指数 | "查询上证指数" |
| getMarketVolatility | 查询波动率 | "A股市场波动率" |
| getMarketSentiment | 查询市场情绪 | "市场情绪怎么样" |
| getSectorPerformance | 查询板块表现 | "科技板块表现" |
| calculatePortfolioReturn | 计算组合收益 | "60%股票、30%债券、10%现金" |
| calculateValueAtRisk | 计算 VaR | "10万投资，95%置信水平" |
| calculateSharpeRatio | 计算夏普比率 | "投资组合的夏普比率" |

**使用场景**:

| 场景 | 推荐值 | 说明 |
|------|--------|------|
| 查询实时数据 | true | 需要调用工具获取数据 |
| 股票分析 | true | 需要股价、历史数据 |
| 简单问答 | false | 不需要工具调用 |
| 知识库问答 | false | 只需要知识检索 |

**示例**:
```bash
# 开启
GET /engine/investment/decide/stream?message=查询苹果股价&enableTools=true

# 关闭
GET /engine/investment/decide/stream?message=什么是夏普比率&enableTools=false
```

---

### 6. enableGraph（可选）

**位置**: 高级选项 → 功能开关 → 工作流图  
**类型**: boolean  
**默认值**: true  

**功能说明**:
```java
enableGraph = true   // 执行完整的工作流程
enableGraph = false  // 跳过工作流步骤
```

**工作流步骤**:
1. 问题感知 - 分析用户需求
2. 知识检索 - 检索财报和报告
3. 数据获取 - 查询实时数据
4. 推理分析 - 基于知识分析
5. 决策生成 - 生成投资建议
6. 流程验证 - 验证完整性

**使用场景**:

| 场景 | 推荐值 | 说明 |
|------|--------|------|
| 完整投资决策 | true | 需要完整工作流 |
| 快速咨询 | false | 跳过工作流提高速度 |
| 简单查询 | false | 不需要完整流程 |
| 详细分析 | true | 需要完整决策流程 |

**示例**:
```bash
# 开启
GET /engine/investment/decide/stream?message=投资建议&enableGraph=true

# 关闭
GET /engine/investment/decide/stream?message=上证指数&enableGraph=false
```

### 参数传递

**文件**: `DecisionEngineView.vue`

```typescript
async function handleSend(message: string) {
  const params: DecisionRequest = {
    message,
    modelName: modelName.value,      // 用户选择的模型
    enableRAG: enableRAG.value,      // 知识检索开关
    enableTools: enableTools.value,  // 工具调用开关
    enableGraph: enableGraph.value,  // 工作流开关
  }

  await startDecision(params)
}
```

### URL 构建

**文件**: `useInvestmentDecision.ts`

```typescript
const url = new URL('/engine/investment/decide/stream', window.location.origin)
url.searchParams.append('message', params.message)

if (params.modelName) url.searchParams.append('modelName', params.modelName)
if (params.threadId) url.searchParams.append('threadId', params.threadId)

url.searchParams.append('enableRAG', String(params.enableRAG ?? true))
url.searchParams.append('enableTools', String(params.enableTools ?? true))
url.searchParams.append('enableGraph', String(params.enableGraph ?? true))
```

## 使用流程

### 1. 打开配置界面
```
点击页面右上角的 "高级" 按钮
```

### 2. 选择模型
```
从下拉框中选择 AI 模型：
- DeepSeek Chat (推荐)
- MiMo 模型
- 通义千问
```

### 3. 配置功能开关
```
勾选/取消勾选对应的功能：
- 知识检索 (RAG)
- 工具调用
- 工作流图
```

### 4. 查看当前配置
```
底部会显示当前配置的预览：
模型: deepSeekChatModel | RAG: 开启 | 工具: 开启 | 工作流: 开启
```

### 5. 发送消息
```
输入消息并点击发送，参数会自动传递给后端
```

## 参数优先级

```
1. URL 查询参数 (@RequestParam) - 最高优先级
2. 请求体 (@RequestBody)
3. 默认值 - 最低优先级
```

**示例**:
```bash
# 即使 POST body 中 enableRAG=true
# URL 中 enableRAG=false 会生效
GET /engine/investment/decide/stream?message=xxx&enableRAG=false
```
