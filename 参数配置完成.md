# 参数配置完成总结

## 已修改的接口

### 1. 流式投资决策接口
```
GET/POST /engine/investment/decide/stream
```

**参数**：
- `message` (必填) - 投资需求
- `modelName` (可选, 默认: deepSeekChatModel) - 模型选择
- `threadId` (可选, 自动生成) - 对话ID
- `enableRAG` (可选, 默认: true) - 知识检索
- `enableTools` (可选, 默认: true) - 工具调用
- `enableGraph` (可选, 默认: true) - 工作流

### 2. 同步投资决策接口
```
GET/POST /engine/investment/decide
```

**参数同上**

---

## 前端调用示例

### 基本调用
```typescript
await startDecision({
  message: '我想投资科技股'
})
```

### 指定模型
```typescript
await startDecision({
  message: '我想投资科技股',
  modelName: 'openAiChatModel'
})
```

### 关闭所有功能（快速查询）
```typescript
await startDecision({
  message: '上证指数是多少',
  enableRAG: false,
  enableTools: false,
  enableGraph: false
})
```

### 只启用工具
```typescript
await startDecision({
  message: '查询苹果公司的股价',
  enableRAG: false,
  enableTools: true,
  enableGraph: false
})
```

## 参数优先级

```
1. URL 查询参数 (@RequestParam)
2. 请求体 (@RequestBody)
3. 默认值
```
